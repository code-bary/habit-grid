# 🟩 HabitGrid — Minimalist Monthly Habit Tracker

**HabitGrid** is a sleek, dark-themed monthly habit tracker designed for those who want to ditch complex spreadsheets. Built with a focus on minimalism and data visualization, it allows users to generate a custom tracking grid in seconds.

[Live Demo](https://code-bary.github.io/habit-grid/)

---

## ✨ Key Features

* **Instant Grid Generation:** No manual setup. Just enter the month and your habits, and the grid is ready.
* **Visual Analytics:** Integrated with **Chart.js** to show daily completion trends and overall progress.
* **Data Persistence:** Uses `localStorage` to save your progress locally in your browser—no login required.
* **Minimalist Design:** A professional dark-mode UI inspired by modern developer tools.
* **Mobile Responsive:** Track your habits on the go from any device.

## 🛠️ Built With

* **HTML5 & CSS3:** For the semantic structure and modern styling.
* **JavaScript (ES6+):** For dynamic grid logic and state management.
* **Chart.js:** For beautiful, interactive progress visualization.
* **AI-Assisted Development:** This project was developed using iterative prompting with **Claude AI**, showcasing a modern "AI-Native" development workflow.

## 🚀 The Build Story (AI-Native Workflow)

Instead of traditional manual coding, this project was built through **5 major iterations** using advanced AI prompting techniques:
1.  **Version 1:** Basic layout and grid generation.
2.  **Version 2-3:** Refining the logic for date handling and persistence.
3.  **Version 4:** Integrating Chart.js for visual data feedback.
4.  **Final Version:** Polishing the UI/UX and adding professional SEO meta tags.
